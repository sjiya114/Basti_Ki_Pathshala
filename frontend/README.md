## Getting Started

1. **Install dependencies**
   ```powershell
   cd frontend
   npm install
   ```
2. **Run the development server**
   ```powershell
   npm run dev
   ```
3. **Build for production**
   ```powershell
   npm run build
   ```

## Customization

- Add new pages in `src/pages/`
- Add reusable components in `src/components/`
- Store images and static assets in `src/assets/`
